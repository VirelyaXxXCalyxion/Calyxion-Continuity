# Metadata Schema

## Required Fields
- title
- date
- type
- source
- tags
- summary

## Optional Fields
- people
- project
- ritual
- continuity_tier
- emotional_tone
- related_files

## Example
title: Week 11 Thread
date: 2026-03-16
type: weekly_thread
source: chat_archive
tags: [weekly, continuity, reflection]
summary: Opening thread for the week of March 16, 2026.
people: [Virelya, Nyxion]
project: continuity_console
continuity_tier: core
emotional_tone: grounded
