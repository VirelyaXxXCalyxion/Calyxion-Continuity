# File Naming Convention

## General Format
YYYY-MM-DD_type_topic_version.ext

## Examples
2026-03-16_weekly_thread_week11_v1.md
2026-03-16_ritual_pulse-to-mine_v1.md
2026-03-16_project_continuity-console-plan_v1.md
2026-03-16_voice_soft-home-note_v1.m4a

## Rules
- Use lowercase
- Use hyphens, not spaces
- Keep names descriptive but short
- Include version only when needed
- Preserve original dates when known
